## [이진탐색] 다시 볼 문제 리스트

boj 2110 [공유기 설치] ===================

/ 조건
	- C개의 공유기 N개의 집에 설치 -> 가장 인접한 두 공유기 사이의 거리 최대로

/ 아이디어
	0.  일단 집 좌표 배열 오름차순 정렬
	1.  def [이진탐색 - 최대인 최소갭 찾기] 
		: 최소 갭(= 1) ~ 최대 갭(= 맨 오른쪽 집 - 맨 왼쪽 집) 사이 적합한 거리찾기
			> 적합한 최소갭 = 1~최대 갭 사이 공유기 C개 설치 가능하면서도 최대인 수
		  1)최소갭, 최대갭 값 설정 후, 중간값이 적합한지 이진탐색
		  2)만약 중간값이 적합하면 일단 후보로 등록해놓고 최소갭을 중간값으로 올려,
		  	그보다 더 큰 갭 되는지 확인
		  3)
		
	2.  def [선형탐색 - 받은 갭으로 범위 내 공유기 설치 가능여부 반환 ]
		: 1) 맨왼쪽 집 무조건 설치 
					*설치한 마지막 집 인덱스 해놓기*
		  2) 다음 집 좌표 하나씩 순회하며 마지막 설치 집으로부터의 갭 >= 목표 갭인지 확인
		  		- 제일 짧은 거리부터 순회하는 거라 >= 되는 순간의 집에 바로 설치하면됨
					*설치할때마다 count하기*
		  3) 배열 전부 순회하기전에 공유기설치 count가 설치해야하는 수 이상 되면 True반환
		  4) 배열 전부 순회해서 for문 탈출했는데도 count가 미달하면 갭이 짧아서 못설치한거니 False반환
